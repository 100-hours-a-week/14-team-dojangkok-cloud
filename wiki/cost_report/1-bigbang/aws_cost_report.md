# AWS 비용 산정 보고서 (V1 Big Bang)

## 1. 개요

본 문서는 도장콕 프로젝트 V1(Big Bang) 버전에서 사용되는 AWS 인프라의 비용 산정 예측 보고서입니다. GCP 인프라(AI 서버) 비용은 별도 문서에서 다룹니다.

### 산정 기간

| 구분 | 기간 | 시간 |
|------|------|------|
| V1 기간 (스프린트 2주 + 릴리즈 1주) | 21일 | 504시간 |
| 월간 기준 | 30일 | 720시간 |

### 환율 및 세금 기준

| 항목 | 적용값 | 비고 |
|------|--------|------|
| 환율 | 1 USD = 1,450 KRW | 2026년 2월 기준 |
| 부가가치세 (VAT) | 10% | AWS 한국 리전 적용 |


## 2. 인프라 현황

### AWS 리소스 구성

| 구분 | 리소스 | 사양 |
|------|--------|------|
| 컴퓨팅 | EC2 Application 서버 | t4g.large (2 vCPU, 8GB RAM, ARM) |
| 컴퓨팅 | EC2 Monitoring 서버 | t4g.medium (2 vCPU, 4GB RAM, ARM) |
| 스토리지 | EBS (Application) | gp3, 100GB |
| 스토리지 | EBS (Monitoring) | gp3, 100GB |
| 스토리지 | S3 버킷 3개 | 데이터 / 배포 아티팩트 / MySQL 백업 |
| 네트워크 | VPC | 10.0.0.0/18 |
| 네트워크 | Internet Gateway | 1개 |
| 네트워크 | VPC Endpoint for S3 | Gateway 타입 |
| 네트워크 | Elastic IP | Application, Monitoring 각 1개 |
| DNS | Route 53 | Hosted Zone 1개 |
| 관리 | SSM (Systems Manager) | Session Manager |

### 무료 리소스

다음 AWS 리소스는 별도 과금이 발생하지 않습니다.

- VPC, Subnet, Route Table, Internet Gateway
- VPC Endpoint for S3 (Gateway 타입)
- Security Group
- IAM (사용자, 그룹, 역할, 정책)
- SSM Session Manager (기본 기능)

### S3 스토리지 사용량 추정

S3 사용량은 서비스 특성을 반영하여 적극적 시나리오 기준으로 산정하였습니다.

**산정 기준**

| 항목 | 값 | 비고 |
|------|-----|------|
| 활성 사용자 수 | 100명 | V1 기간 기준 |
| 사용자당 집노트 수 | 10개 | 노트당 최대 50장 이미지 첨부 가능 |
| 노트당 평균 첨부 이미지 | 25장 | 집 내/외부 촬영 이미지 기준 |
| 이미지 1장 평균 용량 | 4MB | 스마트폰 촬영 사진 (3~5MB 범위) |

**용도별 사용량 산출**

| 용도 | 산출 근거 | 추정 용량 |
|------|-----------|-----------|
| 집노트 이미지 | 100명 x 10개 x 25장 x 4MB | 100GB |
| 계약서 (JPG, PDF) | 100명 x 2건 x 5MB | 1GB |
| MySQL 백업 | 일일 백업 누적분 | 1GB |
| 배포 환경 설정 파일 | .env, application.yaml 등 보안 설정 파일 | 1MB 미만 |
| **합계** | | **약 102GB** |

> 집노트는 사용자가 관심 있는 매물에 대해 내/외부 사진, 하자 확인 사진 등을 노트당 최대 50장까지 첨부할 수 있는 기능입니다. 적극적 시나리오에서는 노트당 평균 25장을 첨부하는 것으로 가정하였습니다.

## 8. 전제 및 참고사항

### 전제 조건

1. 모든 EC2 인스턴스는 **24시간 상시 운영**을 기준으로 산정하였습니다.
2. S3 스토리지 사용량은 적극적 시나리오 기준 **약 102GB**로 추정하였습니다. 산출 근거는 본 문서 2번 "S3 스토리지 사용량 추정" 항목을 참고하시기 바랍니다.
3. 아웃바운드 데이터 전송량은 **월 100GB 이내**로 가정하여 무료 구간으로 산정하였습니다.
4. 환율은 2026년 2월 기준 **1 USD = 1,450 KRW**를 적용하였으며, 실제 청구 시점의 환율에 따라 변동됩니다.

### 산정에서 제외된 항목

- **S3 API 요청 비용** (PUT, GET 등): 소량으로 예상되어 제외
- **Route 53 DNS 쿼리 비용**: 소량으로 예상되어 제외
- **Public IPv4 주소 과금**: 2024년 2월부터 적용된 공인 IPv4 주소 과금($0.005/hour/IP)이 존재하나, 본 산정에서는 제외하였습니다. 2개 IP 기준 월 $7.20 추가 발생 가능성이 있습니다.
- **AWS 프리 티어**: 계정에 프리 티어가 적용되는 경우 실제 비용은 본 산정보다 낮아질 수 있습니다.

### 비용 변동 요인

- 환율 변동
- S3 저장 데이터 증가
- 아웃바운드 트래픽 100GB 초과 시 추가 과금 ($0.126/GB)
- 인스턴스 사양 변경 또는 추가 리소스 도입
