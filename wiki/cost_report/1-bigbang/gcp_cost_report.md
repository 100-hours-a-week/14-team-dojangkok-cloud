# GCP 비용 산정 보고서 (V1 Big Bang)

## 1. 개요

본 문서는 도장콕 프로젝트 V1(Big Bang) 버전에서 사용되는 GCP 인프라(AI 서버)의 비용 산정 예측 보고서입니다. AWS 인프라 비용은 별도 문서에서 다룹니다.

### 산정 기간

| 구분 | 기간 | 시간 |
|------|------|------|
| V1 기간 (스프린트 2주 + 릴리즈 1주) | 21일 | 504시간 |
| 월간 기준 | 30일 | 730시간 |

### 운영 계획

```
Sprint (2주, 14일)     → T4 24/7
Release 월~수 (3일)    → T4 24/7
Release 목~일 (4일)    → L4 24/7
```

### 환율 기준

| 항목 | 적용값 | 비고 |
|------|--------|------|
| 환율 | 1 USD = 1,450 KRW | 2026년 2월 기준 |


## 2. 인프라 현황

### GCP 리소스 구성

| 구분 | 리소스 | 사양 |
|------|--------|------|
| 컴퓨팅 (Sprint) | Compute Engine + T4 GPU | n1-standard-4 (4 vCPU, 15GB RAM) + T4 (16GB VRAM) |
| 컴퓨팅 (Release 후반) | Compute Engine + L4 GPU | g2-standard-4 (4 vCPU, 16GB RAM, L4 24GB VRAM 포함) |
| 스토리지 | Persistent Disk SSD | 200GB pd-ssd |
| 네트워크 | Networking | VPC, 방화벽 등 |
| 관리 | VM Manager | OS 패치 관리 |

> 리전: asia-northeast3 (Seoul), 현재 T4 운영중 -> 목요일부터 L4로 전환

### 무료 리소스

다음 GCP 리소스는 별도 과금이 발생하지 않습니다.

- VPC, Subnet, 방화벽 규칙
- IAM (서비스 계정, 역할, 정책)
- Cloud Console, Cloud Shell

## 8. 전제 및 참고사항

### 전제 조건

1. 모든 GPU VM은 **24시간 상시 운영**을 기준으로 산정하였습니다.
2. Sprint 기간은 T4, Release 후반은 L4로 **GPU 전환 운영**을 가정하였습니다.
3. 환율은 2026년 2월 기준 **1 USD = 1,450 KRW**를 적용하였으며, 실제 청구 시점의 환율에 따라 변동됩니다.

### 비용 변동 요인

- 환율 변동
- 프로모션 크레딧 소진 (현재 -95.21% 적용 중, 한시적)
- GPU 전환 시점 변경 (T4 ↔ L4)
- SUD 할인율 변동 (사용 시간 증가에 따라 최대 ~30%까지 증가)

> 현재 프로모션 크레딧이 적용 중이므로 V1 기간의 실질 비용 부담은 거의 없다. 크레딧 소진 시점을 모니터링하고, 10만원 이하 도달시 계정 변경 검토
